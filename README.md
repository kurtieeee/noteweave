Real-Time Sound-to-Sheet-Music Transcription

Sound to Score is a minimalist web interface that allows users to record or upload musical performances and generate structured sheet music in real time. This frontend connects to a machine learning–based transcription backend.


Features

Record audio via browser microphone

Upload audio files (WAV / MP3)

Playback recorded or uploaded audio

Send audio for transcription

Preview generated sheet music

Download PDF or MusicXML output

Display real-time transcription status


Purpose

This frontend MVP focuses on providing a clean, intuitive, and aesthetically minimal interface for interacting with a real-time automatic music transcription system.


Tech Stack

React + TypeScript
TailwindCSS
Web Audio API
OpenSheetMusicDisplay
REST API integration

Note: This prototype currently includes only the basic features, such as audio recording and file uploading. The transcription functionality has not yet been implemented.
